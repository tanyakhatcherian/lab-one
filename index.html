I like the UI aspects of creating websites and applications a lot but I am interested in learning all tasks involved in app creation. I also love animation but I lack the ability to draw well enough! Maybe one day, with a lot of practice. I have dabbled in one Java course at Carleton but it was a nightmare. I'm hoping HTML & CSS are kinder to me.
